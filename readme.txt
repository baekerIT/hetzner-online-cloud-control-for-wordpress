This Plugin needs an API Token to connect with the Hetzner Cloud API and let you take Control over your Cloud Servers

== Description ==

After activating this Plugin, you only need to activate it and fill in your API Token.

This Plugin allows you to manage your Hetzner Cloud Servers: In the current Version only creating new server instances.

== Installation ==

1. Upload `Hetzner_Cloud_Control_Remote_API.php` to the `/wp-content/plugins/hetzner-online-cloud-control` directory
1. Activate the plugin through the 'Plugins' menu in WordPress?>` in your templates

== Frequently asked questions ==

= What can I do with this Plugin? =

You can create new Server instances in Hetzner Online Cloud. In future Versions more features like Namechange, deletion and more are planned.
